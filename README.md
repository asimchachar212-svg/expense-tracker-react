# Expense Tracker React App

A simple Expense Tracker built with React that allows users to add, delete, and track income and expenses. Data is stored in localStorage so transactions remain available after refreshing the page.

## Features

- Add new transactions
- Delete transactions
- Automatic balance calculation
- Income and expense tracking
- Data persistence with localStorage
- Responsive and clean UI
- React Hooks (useState, useEffect)

## How to Run

1. Clone the repository

```bash
git clone https://github.com/asimchachar212-svg/expense-tracker-react.git
```

2. Navigate into the project

```bash
cd expense-tracker-react
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```
